## ZUL.js

ZUL.js is a data visualizer written in JavaScript and is designed purely for simplicity so anyone can use it and see their data in a way they never have


## Installation
Prerequisites:
Ensure you have Node.js (v18.0.0 or higher) and npm installed on your machine.

Setup:
Clone the repository:


git clone https://github.com/ZUL-js/zul-js.git
cd ZUL.js

Install dependencies:
This command reads the package.json file and fetches all required libraries (Electron, Vite, Mermaid.js, Tailwind CSS).


npm install
Development and Usage

1. Start the Vite server
In your first terminal window, run the development server:


npm run dev

2. Launch ZUL.js
In a second terminal window, launch the Electron desktop application:


npm start
